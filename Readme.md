# Express Starter with TypeScript

Starter project untuk membangun API dengan Express dan TypeScript, menggunakan arsitektur modular dan best practices seperti Service Layer dan Base Controller.

## Fitur

- Express + TypeScript
- Struktur folder modular (controller, service, routes)
- BaseController dengan standar response API
- Prisma ORM (optional, bisa ditambahkan)
- Error handling dan middleware
- Husky pre-commit untuk lint dan type-check
- Handling route not found (404)

---

## Persyaratan

- Node.js >= 16.x
- npm atau pnpm

---

## Instalasi

1. Clone repository

```bash
git clone https://github.com/tajutzam/express_starter_by_zam.git
cd express_starter_by_zam
npm install
npm prisma:migrate
```
